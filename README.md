# Ya_Professional_2021

Участие в олимпиаде 2021

## Getting Started

git clone https://github.com/baby-benz/Ya_Professional_2021

### Prerequisites

Первым делом из терминала запускаем удовлетворение всех зависимостей:

```mvn install -DskipTests```

В *application.properties* указываем желаемое значение для количества символов, подставляемых из content в title:

```
ya-pro.chars-from-title=*количество_символов*
```

После этого собираем .jar-файл и запускаем (установленная версия JRE должна быть не ниже 11):

```
mvn package -DskipTests
java -jar target/lab2-0.1.0.jar
```

### Supported endpoints

Endpoint                                      | Поддерживаемый метод | Действие                                             | MIME-тип возвращаемых данных
--------------------------------------------- | :------------------: | ---------------------------------------------------- | ----------------------------
/swagger-ui/                                  | GET                  | Графический интерфейс Swagger2                       | text/html

REST-контроллер:

Endpoint                                       | Поддерживаемый метод | Действие                                                                                                                                                                                                                        | MIME-тип возвращаемых данных
---------------------------------------------- | :------------------: | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------
/api/notes/{id сущности в базе данных}         | GET                  | Получение списка всех заметок. Если заголовок не указан - то вместо него возвращать первые N символов текста заметки, где N - число, задаваемое в конфигурационном файле                                                        | application/json
/api/notes                                     | POST                 | Добавление заметки с возможностью указания заголовка (title) и текста заметки (content). Заголовок - не обязательный параметр                                                                                                   | application/json
/api/notes                                     | PUT                  | Редактирование заметки по ее идентификатору. Для редактирования доступны: заголовок, текст заметки                                                                                                                              | application/json
